## 1. signup_completed
- 발생 시점: 회원가입 완료 시
- 직접 연결 KPI: 가입 후 7일 재방문율 / 가입 후 7일 첫 복약 기록 전환율 / 가입 후 7일 복약 기록 유지율
- 연결 Funnel 단계: 가입완료

## 2. app_opened
- 발생 시점: 앱 실행 또는 재진입 시
- 직접 연결 KPI: 가입 후 7일 재방문율
- 연결 Funnel 단계: 앱 재진입
- 비고: 첫 복약 기록 전환율과 복약 기록 유지율의 선행 행동으로 볼 수 있음

## 3. medication_registered
- 발생 시점: 약 등록 완료 시
- 직접 연결 KPI: 없음
- 연결 Funnel 단계: 약등록
- 비고: 첫 복약 기록 전환율과 복약 기록 유지율의 선행 단계로 볼 수 있음

## 4. dose_checked
- 발생 시점: 복약 체크 완료 시
- 직접 연결 KPI: 가입 후 7일 첫 복약 기록 전환율 / 가입 후 7일 복약 기록 유지율
- 연결 Funnel 단계: 약 섭취 기록

## 5. report_viewed
- 발생 시점: 리포트 화면 조회 시
- 직접 연결 KPI: 현재 핵심 KPI 기준 없음
- 연결 Funnel 단계: 리포트 조회
