# change-log-v1-to-v2

## 무엇이 바뀌었는가
- v1의 KPI: 복약 기록 유지율 / 기분 기록 완료율 / 리포트 조회 전환율
- v2의 KPI: 재방문율 / 첫 복약 기록 전환율 / 복약 기록 유지율
- v1의 Funnel: 가입 → 캐릭터 선택 → 튜토리얼 → 약 등록 → 첫 복약 체크 완료
- v2의 Funnel: 가입완료 → 앱 재진입 → 약등록 → 약 섭취 기록 → 리포트 조회

## 왜 바꾸었는가
- 신규가입자에게 첫 7일 안에 여러 기능 사용을 기대하는 구조는 다소 무거웠다.
- 초기 단계에서는 재방문, 첫 복약 기록, 반복 사용이 더 중요한 기준이라고 판단했다.
