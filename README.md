## Growth / Product Data Analytics Portfolio

SQL과 Product Analytics 관점에서 사용자 행동 데이터를 분석하고,  
코호트 정의 → 재구매 분석 → CRM 타겟 추출 → 개선 실험 제안까지 연결하는 연습을 정리한 포트폴리오입니다.

## 1. Ecommerce CRM Analysis

BigQuery 공개 데이터셋 `thelook_ecommerce`를 활용해 신규 유저의 첫 구매와 재구매 여부를 분석하는 SQL 프로젝트입니다.

가입 후 30일 이내 첫 구매한 유저를 코호트로 정의하고, 첫 구매 이후 30일 내 두 번째 구매가 발생했는지 확인했습니다.

이후, 첫 구매 이후 재구매하지 않은 유저를 CRM 타겟으로 분류하고, 재구매 유도 메시지나 쿠폰 실험으로 이어질 수 있는 대상군을 정의하였습니다. 

이 프로젝트는 실제 CRM 캠페인을 운영한 경험은 아니지만, 재구매 분석과 CRM 타겟 추출을 위한 SQL 분석 흐름을 연습한 프로젝트입니다.


## 2. DodakTalk Product Growth Case

DodakTalk은 양극성 장애 환자를 대상으로, 복약 기록과 감정 트래킹을 돕는 헬스케어 서비스 기획/개발 프로젝트입니다.

이 프로젝트에서는 실제 운영 로그를 분석하기보다는,신규 사용자의 초기 행동을 어떻게 측정할지에 집중했습니다.

가입 후 7일 재방문율, 첫 복약 기록 전환율, 복약 기록 유지율을 핵심 KPI로 정의하고, 이를 측정하기 위한 이벤트 로그 구조를 설계했습니다.

이 프로젝트는 실제 성과 개선 사례라기보다,초기 서비스 단계에서 KPI와 이벤트 로그를 어떻게 설계할 수 있는지 보여주는 프로젝트입니다.
